# Decodelabs Internship Projects 🚀

This repository contains Python mini projects developed during the Decodelabs Internship Program.  
The projects focus on Python fundamentals, problem-solving, file handling, and command-line application development.

---

# 📌 Projects

## 1. Expense Tracker
A simple Python application to manage daily expenses.

### Features
- Add expenses
- View expenses
- Delete expenses
- Save expense records

---

## 2. Password Generator
A password generator that creates secure random passwords and stores password history.

### Features
- Generate random passwords
- Custom password length
- Password history
- Password strength checker

---

## 3. To-Do List Application
A command-line task management application.

### Features
- Add tasks
- Delete tasks
- Search tasks
- Save tasks permanently

---

# 🛠 Tech Stack

- Python 3
- File Handling
- Functions
- Loops & Conditions
- Random & String Modules
